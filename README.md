# Momentum Catalyst Advisory — Website

Static marketing site for Momentum Catalyst Advisory (MCA LLC).

## Stack

- Plain HTML / CSS, no build step.
- Hosted on Cloudflare Pages, deployed automatically on push to `main` via the Cloudflare Pages GitHub integration.

## Local preview

Open any of `index.html`, `privacy.html`, `terms.html` directly in a browser, or serve from the repo root:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Files

- `index.html` — landing page
- `privacy.html` — privacy policy
- `terms.html` — terms of service
- `styles.css` — shared styles
